## 快速记忆技巧

1. **词根记忆**: trans-(跨越) + form(形式) = transform(变换)
2. **联想记忆**: ellipsis(省略号)像三个点...，表示省略
3. **场景记忆**: throttle(节流)想象水龙头控制水流
4. **对比记忆**: sync(同步) vs async(异步)，a- 表示"非"
