## 技术要点记录

### onBackPress 拦截问题
**问题**：`uni.navigateBack()` 也会触发 `onBackPress`，导致返回被重复拦截

**解决方案**：使用 `options.from` 判断来源
```javascript
onBackPress((options) => {
    if (options.from === 'navigateBack') {
        return false  // 代码调用的返回，不拦截
    }
    // 其他处理...
})
```

### 自定义 Toast 组件用法
```vue
<template>
  <CustomToast ref="toastRef" />
</template>

<script setup>
import CustomToast from '@/components/modals/custom-toast.vue'
const toastRef = ref(null)

// 调用
toastRef.value?.show('提示内容')
</script>
```
