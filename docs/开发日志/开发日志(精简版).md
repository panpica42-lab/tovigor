## 2026-02-27

### 首页控制面板

- **移除原生导航栏**：`pages.json` 添加 `navigationStyle: custom`，首页改为全自定义布局
- **使用指南按钮调整**：从左侧绝对定位改为右侧 flex 布局，与标题对齐
- **新增控制小面板**：
  - 6个图标：亮度、WiFi、音量、蓝牙、设置、开关
  - 圆润长方形外观（小圆角 + 阴影凹凸感）
  - 点击切换状态：深灰色 ↔ 绿色

### 智能评估模块重构

- **目录结构调整**：三大评估模块各自独立为文件夹
  ```
  smartAssess/
  ├── first-skip.vue                  # 入口页（保持根目录）
  ├── body-function-assess/           # 身体机能评估
  ├── body-strength-assess/           # 部位力量评估（原 strength-assess）
  │   ├── body-strength-assess-new.vue
  │   └── components/
  │       └── body-part-card.vue
  └── body-posture-assess/            # 体态姿势评估（新建）
  ```
- **命名规范统一**：所有评估模块统一为 `body-xxx-assess` 三段式命名
- **力量评估入口页**：
  - 新建 `body-strength-assess-new.vue`，旧版保留作为代码资源库
  - 人体模型 + 6个部位卡片布局
  - 抽象 `body-part-card.vue` 组件，显示部位名称、重量、三个子项（热身/训练/挑战）
  - 使用公共返回按钮组件 `CommonBackButton`
- **路由更新**：`pages.json` 和 `first-skip.vue` 跳转路径同步更新
